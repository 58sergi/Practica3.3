Navegador Web

Un navegardor web (en ingles browser) es un sortware o programa que permitia el acceso a la Web,interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcion basica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplo de navegadores

Safari:Safari es un navegador web de codigo cerrado desarrollado por Apple Inc. Está diponible para macOS, IPadOS e iOS.

Mozilla Firefox es un navegador web libre y de codigo abierto desarrollado para Linux, Android, iOS,macOS y Microsoft Windows coordinando por la Corporación Mozilla y la Fundación Mozilla.

Googl Chrome:Google Chrome es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de proyectos de codigo abierto.
